## Implementation Priority

### Tier 1 — Core primitives (build first)
| Component | Why |
|---|---|
| **Buttons** | Used everywhere, foundational CTA |
| **Icons** | Required by buttons, links, cards, forms |
| **Links** | Navigation building block |
| **Tags** | Small, self-contained, used in cards |
| **Chips** | Filter/selection primitive |
| **Toggle** | Simple form control |

### Tier 2 — Form & navigation
| Component | Why |
|---|---|
| **Form (Input Fields)** | Core data entry |
| **Searchbar** | Used in headers and pages |
| **Dropdown** | Used in forms and filters |
| **Accordion** | Content organization |
| **Paginator** | List navigation |

### Tier 3 — Layout components
| Component | Why |
|---|---|
| **Header** | App shell, top-level navigation |
| **Footer** | App shell |
| **Menu Mobile** | Mobile navigation |
| **Header Dropdowns** | Notification & profile menus |
| **List Elements** | Content lists |
| **Table** | Data display |

### Tier 4 — Content & cards
| Component | Why |
|---|---|
| **Cards** (all types) | Primary content display |
| **Widget** | Dashboard content |
| **Calendar** | Date selection |
| **Floating Buttons** | Action triggers |

### Tier 5 — Domain-specific & compositions
| Component | Why |
|---|---|
| **Hero** | Page-level composition |
| **Infographic** | Dashboard visualizations |
| **Chat Items / Blocks / Mockups** | Chat feature |
| **Google Map Elements** | Map feature |
| **Assets & Illustrations** | Static assets (export from Figma) |
